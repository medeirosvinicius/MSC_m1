# Modelo de Preditivo do Impacto das Ciclovias no Município de São Paulo

**Orientadores:** Drª Flavia Sarti e Drº Leandro Totaro  
**Autor do Projeto:** Vinicius Medeiros

O projeto consiste em um sistema probabilístico e preditivo de IMC e Peso baseado em população sintética baseado em Séries Temporais, sendo o modelo desenvolvido em linguagem Python e com framework do Pytorch, entre outras bibliotecas do Python.

O trabalho utilizou de base dados sobre consumo alimentar (calorias por dia), padrão de atividade física (minutos por semana) para prever os pesos, IMC e Nível de Atividade dos indivíduos em um espectro temporal de 6 e/ou 10 anos. Tal modelo aplicado foi aplicado para entender as influências das ciclovias em um contexto dos moradores da cidade de São Paulo e as repercussões sobre os Individuos.

As pastas estão organizadas na ordem:

1. Simulação com Amostra Geral do ISA (2015)  
2. Simulação somente População Exposta às Ciclovias  
3. Simulação Contrafactual I com População Exposta às Ciclovias: Retirada das Ciclovias  
4. Simulação Contrafactual II com População Exposta às Ciclovias: Expansão das Ciclovias  
5. Simulação Contrafactual II com População Exposta às Ciclovias: Expansão das Ciclovias com Política de Equidade  
6. Simulação Contrafactual III com População Exposta às Ciclovias: Abordagem por Categorias/Classe de Individuo (Gênero/Raça/Idade/Condição Socioeconômica/Nível de Instrução)  
7. Simulação Contrafactual III com População Exposta às Ciclovias: Abordagem por Categorias/Classe de Individuo (Gênero/Raça/Idade/Condição Socioeconômica/Nível de Instrução) c/ Política de Equidade Interseccional  
8. Testagem e Comparação do Modelo com Aprendizagem de Máquina via Algoritmo Floresta Aleatória (Random Forest)  
9. Dados Geográficos da CET analisados  
10. Dados Geográficos dos Participantes do ISA (2015-2020)

**Observação:** Nos scripts constam somente os resultados plotados em gráficos; outros dados ou fontes não serão possíveis de acessar via **Lei Geral de Proteção de Dados (LGPD)**.
